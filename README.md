# 班级重大事件管理系统

这是一个简单的班级重大事件管理系统，允许用户添加、编辑、删除和查看班级重要事件。

## 功能特点

- 添加新的班级事件
- 编辑现有事件
- 删除事件
- 按时间顺序显示所有事件
- 响应式设计，支持移动设备

## 技术栈

- 后端：Python Flask
- 前端：HTML, CSS, JavaScript
- 数据库：SQLite
- UI框架：Bootstrap 5

## 安装步骤

1. 安装所需的Python包：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
python app.py
```

3. 在浏览器中访问：
```
http://localhost:5000
```
